# speaki-sh  

ｽﾋﾟｷをあなたのターミナルにいつでも召喚できます  

## 起動方法  
このリポをクローン  
```
sudo chmod +x speaki.sh  
./speaki.sh  
```

コマンドとして設定するには  
`sudo ln -s /path to speaki/speaki.sh /usr/local/bin/speaki`  
で  
`spaeki`  
で呼び出せます  


## 使用法  
起動中に1-9の数字キーを押すと音が流れます  
rを押すと音がランダムに流れます  
gを押すと光ります  
qで停止します
